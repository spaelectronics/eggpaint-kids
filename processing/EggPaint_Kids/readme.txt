EggPaint Kids Edition!


Requirements to run application:
* Sun Java Runtime 6, installed on your system

Additional requirements to run from source code:
* Processing 3 (Download from: http://processing.org/download/  )
* Ani library ( http://www.looksgood.de/libraries/Ani/ )



For operation:
Begin with "Pen Arm" positioned to the LEFT (home) side of the EggBot.
If the "Pen Arm" does not begin in UPPER LEFT corner, press "Home" button, manually move the "Pen Arm" to the left side of the EggBot, and press "OK" to continue.
